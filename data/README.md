# Dataset Overview

This folder contains the raw and processed data for the study on LLM Reasoning Fragility.

## Core Files
- **fragility_research_dataset.csv**: The primary machine-readable dataset used for all analyses in the paper.
- **fragility_k_estimates.csv**: The calculated fragility coefficients ($k$) per model scale.

## Supplementary Materials
- **raw_and_supplementary/**: Contains full workbooks, templates, and Excel versions of the dataset for transparency and auditability.
